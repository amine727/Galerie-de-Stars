# LAB 7 - Galerie de Stars – RecyclerView, Animations et Filtrage

## Description
Application Android affichant une liste de stars avec leurs images et notes.

## Fonctionnalités
- Splash screen avec animation
- Liste avec RecyclerView
- Affichage image + nom + rating
- Utilisation de Glide

- Java
- RecyclerView
- Glide
